## 🔧 Lo que se implementó esta sesión

- Dockerfile + docker-compose con volúmenes persistentes para BD y WhatsApp
- Helmet + rate limiting (excluye ESP32)
- Zod validación en todos los endpoints críticos
- Firebase desde variable de entorno
- Health check endpoint `/health`
- RTSP streaming: snapshots vía ffmpeg desde cámaras Imou/Dahua
- HLS streaming: "VER TRANSMISIÓN EN VIVO" convierte RTSP a streaming web
- Long polling ESP32: comandos desde la nube con latencia <1s
- Sistema de planes de suscripción (crear, precio, duración, MercadoPago)
- Cambio de contraseña para usuarios y admin
- Promover/degradar usuarios a admin
- Registro de actividad con auto-limpieza a 30 días
- Menú hamburguesa responsive en Admin Panel
- Fix registro: no permite acceso sin suscripción activa
- Fix Express 5: wildcard route, body parsing, rate limiter path
- DuckDNS → Cloudflare → Caddy HTTPS automático
